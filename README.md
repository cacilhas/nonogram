# Nonogram

A simple random nonogram game.

## Install

```sh
cargo install --git https://github.com/cacilhas/nonogram --bin nonogram
```

If you have [UPX][] installed, you maybe want to run:

```sh
upx ~/.cargo/bin/nonogram
```
